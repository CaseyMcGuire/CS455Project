  8. You need to have a .htpasswd file stored somewhere on your web server that is not accessible to users. In this file you will store the passwords for all verified users (those who can see the file or directory you're trying to password protect). In another file, .htaccess, you specify the path to the .htpasswd file, the name of the page that is being protected, and the name of the file or files that are being protected (which will be accessed via the protected page).
